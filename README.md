# Online-Course-Shopping-Website

## 專題計畫
### 1. 管理員：
* 管理員可新增、刪除影片，可將舉報的影片下架，也可檢視後台數據庫（顯示售出的圖表統計）
**負責組員：蔡蔓萱 謝宥宣**

### 2.網頁呈現: 
* 公告系統：開學優惠、目前打折、促銷的商品，最新的商品等。
* 優惠活動介紹
* 各年級商品分類
* 搜尋：使用者可搜尋想買的商品。
* 目前線上人數
* 常見問題
* 線上客服
* 聯絡資訊
* 語言
* 幣值
* 購物車：使用者可將想買的商品先放入購物車，使用者可從購物車確認最後購買商品。
* 付款方式、格式、商品收件人
**負責組員：戴育琪 謝宥宣**

### 3.商品呈現：
* 個別商品介紹頁面底下擁有該課程顧客評價與留言
* 影片留言板（星等與評價）
* 產品銷售數量
* 會員價與原價
* 相關商品
* 加購商品推薦
**負責組員：謝宥宣 蔡蔓萱**

### 4.會員功能: 
* 會員有分等級，不同等級有不同待遇（打折數、回扣、生日回饋金等等）。
* 會員註冊
* 顯示會員名稱
* 訂單查詢
* 生日折扣碼
* 歷史訂單查詢
* 願望清單
* 已購買商品
* 觀看期限
* 觀看紀錄
**負責組員：戴育琪 蔡蔓萱**

### 5.額外功能: 
* 課業發問
* 商品開通服務
**負責組員：全體組員**

註：使用Github與VSCode共同編輯
